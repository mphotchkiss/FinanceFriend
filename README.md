# FinanceFriend
1. Install homebrew and python: https://docs.python-guide.org/starting/install3/osx/#install3-osx
2. Install tkinter: `brew install python-tk@3.9`
3. Install all of the following dependencies using `pip install`
- beautifulsoup4
- Pillow
- numpy
- matplotlib
- ofxparse
- tkcalendar
- google
